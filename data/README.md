# Data Availability

## Dataset Access

The dataset used in this project is **not publicly available**.

This analysis was conducted using data representing real-world hospital operations. Although identifying information has been removed and the hospital has been anonymized, the dataset contains sensitive patient-level clinical and operational information.

To protect patient confidentiality, institutional privacy, and comply with responsible healthcare data practices, the original datasets have not been included in this public repository.

## Dataset Structure

The project was built using four relational datasets:

- **Patient Details** – Patient demographics and baseline information.
- **Admission Details** – Hospital admissions, diagnoses, discharge outcomes, and admission records.
- **Doctors** – Physician information and clinical specialties.
- **Risk Factors** – Patient lifestyle and behavioral risk factors.

These tables were integrated within PostgreSQL to support the analyses presented in this repository.

> **Note:** The SQL queries, project documentation, methodology, dashboard, and analytical outputs are fully available within this repository, allowing the analytical workflow to be reviewed even though the source data cannot be publicly distributed.
