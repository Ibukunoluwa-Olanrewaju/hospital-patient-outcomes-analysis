# SQL Queries

## Query Library

This folder contains the SQL queries used to transform the raw hospital data into meaningful analytical insights.

The queries document the analytical workflow, including database table creation, relational joins, Common Table Expressions (CTEs), and the analytical queries used to answer the project's business questions.

### Query Categories

- **Table Creation** – SQL scripts used to create the database tables for the imported datasets.
- **Table Joins** – Queries used to integrate the Patient Details, Admission Details, Doctors, and Risk Factors tables into a relational dataset.
- **Common Table Expressions (CTEs)** – Structured queries used to simplify complex analyses and improve query readability.
- **Analytical Queries** – Queries used to calculate mortality rates, DAMA rates, disease prevalence, physician workload, and other project metrics.

The SQL scripts demonstrate how PostgreSQL was used to transform relational healthcare data into actionable insights for hospital performance evaluation.
